Игра-стрелялка с несколькими уровнями сложности, где полицейскому-коту будет необходимо ходить на задания и побеждать плохих парней.

Управление происходит так:
   D, →  -  движение вправо
   A, ←  -  движение влево
   W, ↑  -  движение вверх
   S, ↓  -  движение вниз

Для запуска скрипта выполни python main.py

презентацию к игре
